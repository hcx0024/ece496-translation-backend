## What Is the Fallback?

When we look up a word to get example sentences, we first ask an online dictionary.  
**If the dictionary has no examples (or we can’t reach it), we don’t leave the user with nothing.**  
We automatically create **3 simple example sentences** so the user always sees the word in context.

---

## When Does It Happen?

- The word isn’t in the dictionary (e.g. rare word, name, or typo).
- The dictionary service is slow or temporarily unavailable.
- The word exists but has no example sentences.

In all these cases, we **still show 3 example sentences** instead of zero.

---

## How Does It Work? (In Plain Words)

1. We try the dictionary first.
2. If we get no examples, we switch to “fallback” mode.
3. We guess whether the word is a **thing** (noun), **action** (verb), or **description** (adjective) from how it’s spelled.
4. We plug the word into **simple, natural sentence templates** for that type:
   - **Things:** e.g. “I saw a [word] yesterday.” / “Can you show me the [word]?”
   - **Actions:** e.g. “I [word] every day.” / “Can you [word]?”
   - **Descriptions:** e.g. “This is very [word].” / “I feel [word] today.”
5. We create **3 different** examples so the user sees variety.
6. Those 3 sentences are translated to the chosen language, just like dictionary examples.

So: **dictionary when possible, friendly generated sentences when not** — and the user always gets something useful.

---

## Why It Matters for Users

- **No empty screens:** Users always get example sentences.
- **Still useful:** Even simple examples show how the word is used.
- **Same flow:** Translation and pronunciation work the same whether examples came from the dictionary or from the fallback.
- **Transparent:** In the app, we can label examples as “from dictionary” or “generated” so it’s clear where they came from.

---

## One-Sentence Summary

**We always show example sentences: when the dictionary has them we use those, and when it doesn’t we generate 3 simple, contextual sentences so the user never sees an empty result.**
